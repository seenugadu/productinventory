# productinventory
Integration micro service to add, update and get product inventories

#Endpoints
To get all product inventories : http://localhost:8082/productinventories
To add/update a product inventory : http://localhost:8082/addproductinventory
To get a product inventory based on product and store : http://localhost:8082/productinventory?sku=xxx&storeNbr=xxxx
